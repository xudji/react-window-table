# 基于 react-window 封装虚拟滚动大数据渲染 table 组件

# Development

```bash
# install dependencies
$ npm install

# start dev server
$ npm start

# build docs
$ npm run build
```

# npm 发布流程

```
# 发布一个 patch 版本
$ npm version patch -m "build: release %s"

# 体检 + build
$ npm run prepublishOnly

# 发布
$ npm publish
```
